# AiNiee 7.3 版本更新日志

## 新增功能与优化
1. **独立 QA 评估页面**：
   - 将“双语比对分析（QA评估）”从原来的主提取流程中抽离，现已移至“提取、翻译、校润”右侧的新独立标签页中。
   - 优化了导入流程：支持从本地导入已翻译的 `.docx` 或 `.pdf` 文件，并与项目中的原文自动进行智能对齐。

2. **多线程异步 QA 评估**：
   - 新增了“开始 QA 评估”和“停止 QA 评估”按钮。
   - 新增了“并发线程”设置项，用户可自行指定并发评估的线程数。
   - 对底层的并发线程池（`ThreadPoolExecutor`）进行了彻底的异步化重构，现在可以**随时瞬间停止**并且能够以10条为单位实时刷新结果，解决了以前批量评估时导致的界面卡顿与死机问题。

3. **智能清理特殊标签**：
   - 针对某些文档（特别是部分 Word 提取件）中含有大量 `<>` 占位符标签的情况，新增了“移除 `<>` 等特殊标签”的一键清洗功能。

4. **原生的表格自适应换行与高度支持**：
   - 引入了 `WordWrapDelegate`（继承自 `qfluentwidgets` 的 `TableItemDelegate`），让数据表格首次完美支持纯文本的自动换行展示。
   - 新增“展开选中行”和“收起选中行”的功能，能够自适应原文、译文或QA结果中最长的一列动态调整行高，解决了文本超长显示不全的问题。

5. **界面与主题样式深度兼容**：
   - 完美修复了自适应换行时的文字重叠（双层字）问题。
   - 修复了换行模式下深浅主题切换时颜色不跟随的问题。
   - QA 评估结果文字状态（成功为绿色，瑕疵为红色）现在支持自适应换行并与其他样式保持一致的字体大小与粗细。
