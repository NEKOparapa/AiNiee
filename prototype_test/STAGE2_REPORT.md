## 🔍 技术实现细节

### 格式标记转换
`python
# 添加标记
"EC50" + [format@2-4:subscript] → "EC_{50}"

# 解析标记  
"EC_{50}" → "EC50" + [format@2-4:subscript]
`

### XML回写策略
`xml
<!-- 有格式的文本切分成多个 run -->
<w:p>
  <w:r><w:t>EC</w:t></w:r>
  <w:r>
    <w:rPr><w:vertAlign w:val="subscript"/></w:rPr>
    <w:t>50</w:t>
  </w:r>
</w:p>
`
